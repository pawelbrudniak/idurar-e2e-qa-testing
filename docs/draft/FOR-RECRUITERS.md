## 🧰 Do I have experience with Jira / TestRail?

👉 **Yes --- at the process level.**

While this project does not use Jira or TestRail directly, it replicates
their core functionalities:

-   Issue tracking (Jira equivalent)
-   Test case management (TestRail equivalent)
-   Workflow management
-   Defect lifecycle handling
-   Traceability across all testing levels

------------------------------------------------------------------------

## 🔄 What exactly is implemented?

The project includes:

-   Structured backlog (Epics, Stories, Test Cases, Bugs)

-   Workflow stages (Backlog → In Progress → Testing → Done)

-   Defect lifecycle (New → In Progress → Fixed → Retest → Closed →
    Reopened)

-   Full traceability:

    Epic → Story → Test Case → Bug

-   Linking between all work items

-   Real test artifacts and bug reports
