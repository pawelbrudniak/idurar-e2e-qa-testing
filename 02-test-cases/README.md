# 🧪 Test Cases

This directory contains designed test cases for the iDURAR system.

## Approach

Test cases are based on real user scenarios and system behavior observed during testing and investigation.

Each test case includes:
- objective
- preconditions
- test data
- steps
- expected result
- actual result
- status

## Naming Convention
TC-[AREA]-[ID]

Example:
TC-LOGIN-01
